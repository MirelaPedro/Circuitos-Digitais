# Circuitos Digitais
Este é o repositório que salvo todos os trabalhos/projetos realizados em circuitos digitais

## Projetos
### Calculadora Digital
O referido trabalho foi desenvolvido com o objetivo de projetar uma calculadora digital que realize as operações mais básicas da matemática (soma, subtração, multiplicação e divisão) a partir da escolha do usuário. A calculadora será projetada para a entrada de 4 bits clássicos e utilizará displays de sete segmentos para mostrar o resultado ao usuário.
